    +   作业

        完成精准 import 转码：

        ```js
        import { Select as MySelect, Pagination } from 'xxx-ui';
        // import UI2 from 'xxx-ui';
        import * as UI from 'xxx-ui';
        ```

        到 

        ```js
        import MySelect from 'xxx-ui/Select/Select.js';
        import Pagination from 'xxx-ui/Pagination/Pagination.js';
        // import UI2 from 'xxx-ui';

        import * as UI from 'xxx-ui';
        ```

        **类似即可，自由发挥**
